A simple and responsive **Student Profile Card** web application built using **HTML, CSS, and JavaScript**. The application allows users to enter student details, automatically calculates the student's grade based on marks, determines the pass/fail status, and displays the information in a stylish profile card.

---

## 🚀 Features

* 📝 Enter Student Name
* 🆔 Enter Roll Number
* 📊 Enter Student Marks
* ✅ Input Validation
* 🎯 Automatic Grade Calculation
* 📈 Pass/Fail Result
* 🟢 Pass Result displayed in Green
* 🔴 Fail Result displayed in Red
* 🎨 Modern Responsive UI
* 📱 Mobile-Friendly Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ How It Works

1. Enter the student's name.
2. Enter the roll number.
3. Enter marks (0–100).
4. Click the **Display Profile** button.
5. The application will:

   * Validate the input.
   * Calculate the grade.
   * Determine Pass or Fail.
   * Display the student's profile card.

---

## 🎯 Grade System

|    Marks | Grade | Result |
| -------: | :---: | :----: |
| 90 - 100 |   A+  |  Pass  |
|  80 - 89 |   A   |  Pass  |
|  70 - 79 |   B   |  Pass  |
|  60 - 69 |   C   |  Pass  |
|  50 - 59 |   D   |  Pass  |
| Below 50 |   F   |  Fail  |
